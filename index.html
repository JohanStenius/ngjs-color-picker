ngjs-color-picker

A small directive which essentially is a color picker but with a few extra functions - for AngularJS. Based on ng-color-picker.

Requirements: Angular 1.2+

Features

  • Customize the appearance of the color picker
  • Generate random colors
  • Generate gradient colors

Demo

  • Demo
  • Plunker (Note: All functions are not working properly on Plunker)

Installation

via bower:

$ bower install --save ngjs-color-picker

manually:

Clone or download the repository and include the js and css files in your app.

JS

<script type="text/javascript" src="js/ngjs-color-picker"></script>`

CSS

<link rel="stylesheet" href="css/ngjs-color-picker.css" />

Inject the directive as a dependency in your app:

angular.module('myApp', ['ngjsColorPicker']);
